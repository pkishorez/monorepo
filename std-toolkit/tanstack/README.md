# @std-toolkit/tanstack

TanStack DB integration with evolving schema (eschema)

## Installation

```bash
pnpm add @std-toolkit/tanstack effect
```

## Usage

```typescript
import { hello } from "@std-toolkit/tanstack";

console.log(hello());
```

## Development

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test

# Run tests once
pnpm test:run

# Type check
pnpm lint

# Build
pnpm build

# Watch mode
pnpm dev
```
