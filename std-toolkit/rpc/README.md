# rpc

RPC related utilities for STD toolkit

## Installation

```bash
pnpm add rpc effect
```

## Usage

```typescript
import { hello } from "rpc";

console.log(hello());
```

## Development

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test

# Run tests once
pnpm test:run

# Type check
pnpm lint

# Build
pnpm build

# Watch mode
pnpm dev
```
