`use-effect-ts` is a set of React hooks for running Effect.TS programs inside React components, bridging Effect's runtime with React's rendering lifecycle.

## Install

```sh
npm install use-effect-ts effect react
```

## Core concepts

- **Hooks over Effects** — run and observe Effect programs from within React components using idiomatic hooks.
- **Effect-native** — composes with the wider Effect ecosystem, so your data-fetching and side effects stay in Effect.
